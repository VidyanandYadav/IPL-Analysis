# 📧 Email Spam Classifier

A simple web app built with **Streamlit** that classifies emails or messages as **Spam** or **Not Spam** using a machine learning model.

## 🛠 Features

- ✅ Detects spam and non-spam emails.  
- ✅ Preprocesses text: lowercase, tokenization, stopword removal, punctuation removal, and stemming.  
- ✅ Displays prediction with clear indicators:  
  - 🚨 Spam  
  - ✅ Not Spam  
- ✅ Works for short messages and longer emails.  
- ✅ Compatible with Python 3.12+ and Streamlit Cloud.
